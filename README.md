Ansible Role used for install and configure Jira
================================================

This role will install and configure Atlassian Jira on target systems.

Requirements
------------

None.

Role Variables
--------------

None.

Dependencies
------------

None.

Example Playbook
----------------
```
- hosts: all
  roles:
    - role: jira
```
